# 圖片文字移除工具 (Image Text Removal Tool)

[繁體中文](#繁體中文) | [简体中文](#简体中文) | [English](#english) | [日本語](#日本語) | [한국어](#한국어)

## 繁體中文

### 功能介紹
這是一個簡單易用的網頁工具，可以幫助您快速移除圖片中的文字。支援多種選擇工具和顏色處理選項，讓您能夠精確地移除圖片中的文字內容。

### 主要功能
- 支援多種圖片格式上傳
  - 支援 PNG、JPG、JPEG、GIF 等常見圖片格式
  - 自動優化上傳圖片大小，確保最佳處理效果
  - 支援拖放上傳功能

- 提供矩形選擇和套索工具
  - 矩形選擇：適合處理規則形狀的文字區域
  - 套索工具：可自由繪製不規則形狀，適合複雜文字區域
  - 支援選區調整和移動

- 顏色拾取器功能
  - 精確選取圖片中的特定顏色
  - 可調整顏色容差範圍
  - 支援多點顏色採樣

- 透明背景支援
  - 可將移除區域設為完全透明
  - 支援半透明效果
  - 保留原始圖片透明度

- 操作歷史記錄（撤銷功能）
  - 支援多步驟撤銷
  - 可恢復到任意操作步驟
  - 自動保存操作歷史

- 多語言界面支援
  - 支援繁體中文、簡體中文、英文、日文、韓文
  - 自動檢測瀏覽器語言
  - 可手動切換界面語言

### 使用方式
1. 點擊「上傳圖片」按鈕選擇需要處理的圖片
2. 使用選擇工具（矩形或套索）標記要移除的區域
3. 選擇填充顏色（預設為透明）
4. 點擊「清除選區」完成文字移除
5. 使用「下載圖片」保存處理結果

### 技術特點
- 純前端實現，無需後端服務
- 響應式設計，支援各種設備
- 直覺的使用者界面
- 即時預覽效果

## English

### Introduction
A user-friendly web tool for removing text from images. It offers various selection tools and color processing options for precise text removal.

### Key Features
- Multiple image format support
  - Supports PNG, JPG, JPEG, GIF and other common formats
  - Automatically optimizes uploaded image size
  - Supports drag and drop upload

- Rectangle and Lasso selection tools
  - Rectangle selection: ideal for regular-shaped text areas
  - Lasso tool: free-form selection for complex text areas
  - Supports selection adjustment and movement

- Color picker functionality
  - Precise color selection from images
  - Adjustable color tolerance range
  - Supports multiple color sampling points

- Transparent background support
  - Set removed areas to fully transparent
  - Supports semi-transparent effects
  - Preserves original image transparency

- Operation history (undo feature)
  - Multi-step undo support
  - Restore to any previous operation
  - Automatic operation history saving

- Multi-language interface
  - Supports Traditional Chinese, Simplified Chinese, English, Japanese, Korean
  - Automatic browser language detection
  - Manual language switching option

### How to Use
1. Click "Upload Image" to select your image
2. Use selection tools (Rectangle or Lasso) to mark areas
3. Choose fill color (default: transparent)
4. Click "Clear Selection" to remove text
5. Use "Download Image" to save the result

### Technical Features
- Frontend-only implementation
- Responsive design
- Intuitive user interface
- Real-time preview

## 한국어

### 기능 소개
이미지에서 텍스트를 쉽게 제거할 수 있는 웹 도구입니다. 다양한 선택 도구와 색상 처리 옵션을 제공하여 이미지 내의 텍스트를 정확하게 제거할 수 있습니다.

### 주요 기능
- 다양한 이미지 형식 지원
  - PNG, JPG, JPEG, GIF 등 일반적인 형식 지원
  - 업로드 이미지 크기 자동 최적화
  - 드래그 앤 드롭 업로드 지원

- 사각형 선택 및 올가미 도구
  - 사각형 선택: 규칙적인 형태의 텍스트 영역에 적합
  - 올가미 도구: 복잡한 텍스트 영역의 자유로운 선택 가능
  - 선택 영역 조정 및 이동 지원

- 색상 선택기 기능
  - 이미지에서 정확한 색상 선택
  - 색상 허용 범위 조정 가능
  - 다중 포인트 색상 샘플링 지원

- 투명 배경 지원
  - 제거 영역을 완전 투명으로 설정 가능
  - 반투명 효과 지원
  - 원본 이미지 투명도 유지

- 작업 기록 (실행 취소 기능)
  - 다중 단계 실행 취소 지원
  - 임의의 작업 단계로 복원 가능
  - 작업 기록 자동 저장

- 다국어 인터페이스
  - 번체 중국어, 간체 중국어, 영어, 일본어, 한국어 지원
  - 브라우저 언어 자동 감지
  - 수동 언어 전환 가능

### 사용 방법
1. "이미지 업로드" 버튼을 클릭하여 이미지 선택
2. 선택 도구(사각형 또는 올가미)로 제거할 영역 표시
3. 채우기 색상 선택(기본값: 투명)
4. "선택 영역 지우기" 클릭하여 텍스트 제거
5. "이미지 다운로드"로 결과 저장
